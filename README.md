# Прогнозирование наличия сердечно-сосудистых заболеваний на основе характеристик пациентов

## Введение

В статье [An artificial intelligence model for heart disease detection using machine learning algorithms](https://www.sciencedirect.com/science/article/pii/S2772442522000016) раскрывается тема применения методов машинного обучения для прогноза сердечно-сосудистых заболеваний.  В научной публикации  разработано приложение на языке программирования Python для  для мониторинга состояния здоровья. Для приложения использован классификатор на основе метода случайного леса. 

Нам были даны [данные](https://www.kaggle.com/competitions/tech-weekend-data-science-hackathon/data), содержащие информацию из медицинских карточек. 

## Цель и задачи исследования

Основная цель: обучить минимум три модели машинного обучения, включая нейронную сеть, для получения прогноза развития заболевания сердечно-сосудистой системы. 

Постевлены следующие задачи:

1. Провести предподготовку данных и первично оценить данные.

2. Провести исследовательский анализ данных, очистить данные от грубых выбросов и оценить возможные пропуски.

3. Предподготовить данные для моделей.

4. Обучить минимум три модели: логистическую регрессию, случайный лес, нейронную сеть.

5. Сделать общие выводы и сделать прогноз на тестовых данных с помощью лучшей модели.

## Выводы

Как видно из диаграмм, что в данных примерно половила людей с заболеванием и без. Но в основном показатели крови в норме. В описании данных не была указана расшифровка для колонки "sex". То есть мы не знаем точное распределние представителей разного пола в датасете, но большую часть данных представляют люди, зашифрованные под 1 (предположительно 1 - мужчина, 0 - женщины).
Распределение показателей не имеет сильных отличий между группами с заболеванием и без поставленого диагноза. Внутри данных не обнаружено мультиколинеальность. В качестве метода подготовки данных был выбран метод StandardScaler. 

Обучили три типа: логистическую регрессию, случайный лес, нейронную сеть. Первые две модели были обучены на базовой уровне и с подбором гиперпараметров. 

По итогу были получены высокие метрики точности  ROC-AUC как при кросс-валидации, так и при правильности. Это может указывать на переобучение моделей. В сязи с этим для улучшения моделей необходимо получить так же информацию целевого признака для тестового набора данных.
