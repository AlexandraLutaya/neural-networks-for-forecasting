# Прогнозирование наличия сердечно-сосудистых заболеваний на основе характеристик пациентов

## Введение

В статье [An artificial intelligence model for heart disease detection using machine learning algorithms](https://www.sciencedirect.com/science/article/pii/S2772442522000016) раскрывается тема применения методов машинного обучения для прогноза сердечно-сосудистых заболеваний.  В научной публикации  разработано приложение на языке программирования Python для  для мониторинга состояния здоровья. Для приложения использован классификатор на основе метода случайного леса. 

Нам были даны [данные](https://www.kaggle.com/competitions/tech-weekend-data-science-hackathon/data), содержащие информацию из медицинских карточек. 

## Цель и задачи исследования

Основная цель: обучить минимум три модели машинного обучения, включая нейронную сеть, для получения прогноза развития заболевания сердечно-сосудистой системы. 

Постевлены следующие задачи:

1. Провести предобработку данных и первично оценить данные.

2. Провести исследовательский анализ данных, очистить данные от грубых выбросов и оценить возможные пропуски.

3. Предподготовить данные для моделей.

4. Обучить модели, сделать общие выводы и сделать прогноз на тестовых данных с помощью лучшей модели.

## Выводы

1. В данных около 81 % пациентов без меланомы, около 87 % пациентов без себорейного кератоза. Из всех выборки пациентов 68 % - это пациенты с невусом.

2. По средней статистике изображений можно сделать вывод, что изображения без засветов, но по стандартным отклонениям ширины и высоты видно, что фотографии загружены разного размера.

3. Обучили две модели TensorFlow и ResNet, данные предварительно прошли аугментацию.

Правильность получили для обеих моделей на обучающей выборке 0.74 для TensorFlow и 0.71 для ResNet, на тренировочных - 0.71 и 0.71 соотвественно.

4. На данный момент кандидатом для дальнейшей доработки является TensorFlow.
